# AWS Labs — DevOps & AI Infrastructure

A hands-on learning journey from AWS fundamentals to AI deployment infrastructure.

Built by a DevOps engineer deliberately transitioning into AI infrastructure roles.

---

## 🎯 Goal

Work through each lab in order, building toward a fully functional AI deployment pipeline on AWS. Every lab builds on the last.

## 🛠️ Prerequisites

- AWS free tier account (aws.amazon.com/free)
- AWS CLI installed and configured
- Docker Desktop installed
- kubectl installed
- Basic terminal/command line comfort

---

## 💡 How to use these labs

Each lab has step-by-step instructions, what to expect, common errors, and reflection questions. Don't just follow the steps — read the reflection questions and actually think about them. That's what turns labs into interview stories.
